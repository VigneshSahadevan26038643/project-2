# Project 2 

A collection of web development tasks showcasing modern UI/UX design, interactive elements, and frontend logic.

## 📂 Project Structure
- **/task 1** - Neon Beats Web Music Player
- **/task 2** - Ascend Fitness Premium Gym Landing Page

---

## 🎵 Task 1: Neon Beats Web Music Player

A beautifully designed, highly interactive web-based music player with premium deep blue neon aesthetics and glassmorphism.

### Features
- **Dynamic Aesthetic Design:** Modern UI designed with smooth gradients and glass-panel effects.
- **Full Playback Controls:** Seamless play, pause, next, and previous functionalities.
- **Interactive Progress & Volume:** Custom designed neon sliders for track progress tracking and volume adjustments.
- **Dynamic Playlist:** Real-time queue view populated by JavaScript.

---

## 🏋️ Task 2: Ascend Fitness Gym Website

A single-page scrollable gym website designed with a dark, premium aesthetic and animated interactive elements.

### Features
- **Modern Landing Page:** Hero section, facilities grid, services overview, and layered membership plans.
- **Interactive Registration Modal:** A fully designed overlay modal for new member sign-ups accessed via the website.
- **Dynamic Scroll & Navigation:** Intercepted anchor tags for smooth scrolling, mobile drawer menu, and sticky nav functionality.
- **Premium Aesthetics:** Vibrant accents, glassmorphism overlays, and hover micro-animations.

---

## 💻 Tech Stack
- **HTML5:** Semantic architecture.
- **CSS3:** Custom design system without standard frameworks, utilizing dynamic variables, micro-animations, and flexbox layouts.
- **Vanilla JavaScript:** Fast, dependent-free execution managing state, DOM updates, and interactive UI logic.
