# Project 2 - Neon Beats Web Music Player

A beautifully designed, highly interactive web-based music player with premium deep blue neon aesthetics and glassmorphism.

## 📂 Project Structure
- **/task 1** - Contains the isolated web music player application.

## 🎵 Features
- **Dynamic Aesthetic Design:** Modern UI designed with smooth gradients and glass-panel effects.
- **Full Playback Controls:** Seamless play, pause, next, and previous functionalities.
- **Interactive Progress & Volume:** Custom designed neon sliders for track progress tracking and volume adjustments.
- **Dynamic Playlist:** Real-time queue view populated by JavaScript.

## 💻 Tech Stack
- **HTML5:** Semantic architecture.
- **CSS3:** Custom design system without standard frameworks, utilizing dynamic variables, micro-animations, and flexbox layouts.
- **Vanilla JavaScript:** Fast, dependent-free execution managing audio state, DOM updates, and custom controls.
